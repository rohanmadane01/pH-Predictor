📚 ABOUT THE PROJECT

The pH Value Predictor is an interactive chemistry-based web project used to estimate the pH value of acids, bases and neutral solutions. The project uses mathematical and logarithmic formulas of chemistry to determine whether a solution is acidic, basic or neutral.


⚗️ WHAT IS pH?

pH stands for “Potential of Hydrogen”. It measures the concentration of hydrogen ions (H⁺) present in a solution. The pH scale ranges from 0 to 14.

  • pH < 7 → Acidic
  • pH = 7 → Neutral
  • pH > 7 → Basic

🧪 FORMULAS USED

Strong Acid: pH = -log[H⁺]

Strong Base: pH = 14 - pOH

Weak Acid/Base: Uses equilibrium constants Ka and Kb.
